## Getting Started

### Prerequisites

- yarn

### Running

1. Clone the repo

```sh
git clone https://github.com/Toshiuk/r5go-b2w-front.git
```

2. Install NPM packages

```sh
yarn install
```

3. Run development mode

```sh
yarn dev
```
